# Andrey Portfolio

База для портфолио на `Next.js`.

Пока здесь нет верстки по Figma. Сейчас это чистый фундамент:
- главная страница
- отдельные адреса для будущих кейсов
- страница `Стена`
- переключение языка `ru/en`
- отдельные файлы для текстов
- понятная структура для будущих кейсов, медиа и ссылок

## Как запустить

```bash
npm install
npm run dev
```

Открыть в браузере:

```text
http://localhost:3000
```

Проверки:

```bash
npm run lint
npm run build
```

## Как устроен проект

`src/app`
- страницы сайта
- `page.tsx` это главная
- `cases/[slug]/page.tsx` это шаблон отдельного кейса
- `wall/page.tsx` это страница `/wall`
- `layout.tsx` это общая оболочка сайта

`src/components`
- отдельные блоки сайта
- шапка
- переключатель языка
- содержимое главной
- содержимое страницы `Стена`

`src/data`
- тексты и данные сайта
- русский и английский язык
- список кейсов
- контакты

`src/types`
- типы данных
- простыми словами: правила, которые помогают коду не путаться

`public`
- статические файлы
- сюда потом кладём `CV`, изображения, gif, видео

## Куда класть реальные файлы

`public/cv`
- сюда потом кладём PDF с резюме
- пример: `public/cv/andrey-carev-cv.pdf`
- страница `/cv` автоматически покажет PDF, если файл лежит по этому пути

`public/media`
- сюда потом кладём картинки, gif и видео

`public/media/wall`
- медиа для страницы `Стена`
